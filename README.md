# Enterprise LAN Design and Simulation using Cisco Packet Tracer

## Overview
This project presents the design and simulation of an enterprise-level Local Area Network (LAN) using Cisco Packet Tracer. The network is structured to demonstrate efficient communication, scalability, and logical segmentation using modern networking concepts.

## Objectives
The main objective of this project is to design a reliable and scalable LAN architecture that supports multiple departments while ensuring proper traffic management, security, and performance optimization.

## Network Design
The network is designed using a structured approach that includes routers, switches, and end devices. Logical segmentation is achieved through VLAN configuration, allowing different departments or user groups to operate within isolated network segments.

## Key Features
- VLAN-based network segmentation  
- Inter-VLAN communication using routing  
- Structured IP addressing scheme  
- Router and switch configuration  
- End-to-end connectivity testing  
- Scalable and modular network design

## How to Use
1. Open the `.pkt` file using Cisco Packet Tracer  
2. Explore the network topology  
3. Check device configurations (routers/switches)  
4. Test connectivity using ping and simulation mode
